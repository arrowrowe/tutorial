tqChart (Demo @ Tongqu) 是基于 ECharts (Demo) 封装的 angular directive.

ECharts 本身是相当强的 js 图表, 不过同去里使用时觉得写起来并不太顺手, 于是封装.

tqChart 是若干种图表的结合, 为了比较方便的调用 ECharts 来初始化, 又写了 tqChartE, 和 tqSlider 一样用 element 提供初始化所需 DOM. 

var chartTestRows = [
    {user_name: 'arrowrowe', info0: '咩!', info1: [0, 1, 2, 3]},
    {user_name: 'arrow', info0: '咩!', info1: [1, 3]},
    {user_name: 'sway', info0: '怎么了', info1: [3, 4]},
    {user_name: 'at7', info0: '还好吧', info1: [2, 3]},
    {user_name: 'at15', info0: '还好吧', info1: []},
    {user_name: 'pvguo', info0: '还好吧', info1: [0, 2]},
    {user_name: '路人甲'}
];

$scope.chartSingle = {
    description: 'tqChart 效果示例',
    defaultChartType: 'e-pie',
    graphWidth: 968,
    graphHeight: 300,
    rowNameField: 'user_name',
    rowKeyField: 'info0',
    rows: chartTestRows
};

封装中遇到一个问题, ECharts 要求被初始化的 DOM 具有绝对宽高. 一开始是指定 graphWidth 和 graphHeight 来应对, 但后来在 $modal 中用 tqChart 时发现不能写死宽高了, 于是在 tqChart 里为 tqChartE 设置宽高.
